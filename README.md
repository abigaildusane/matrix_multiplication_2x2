## Matrix Multiplication (2×2)
Python program to multiply two 2×2 matrices entered by the user.

## How to Run
python matrix_multiplication.py

## Usage
Enter values for Matrix A
Enter values for Matrix B
Each row must contain 2 numbers separated by spaces
Program prints the result matrix

## Example
Enter matrix A:
Row 1: 3 4
Row 2: 4 4

Enter matrix B:
Row 1: 4 4
Row 2: 4 4

Result:
|28, 28|
|32, 32|

## Formula
If
A = | a b |
    | c d |
B = | e f |
    | g h |
Then
A × B = 
| ae + bg af + bh |
| ce + dg cf + dh |
